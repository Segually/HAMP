# HAMP

A clean-room reimplementation of Hybrid Animals' Friend Server (and later, Game Servers).

## How to use

Download HAModHelper, edit the IP in the connection hijacker to your server's IP that will run the build.
Open port 7002.
Compile and run the project. If all goes well, you will see it connect to port 7002.
Open the game with the compiled HAMH build and try to open your Friends menu.
